# DAFTAR LAYANAN AI & FITUR TERRY (v2.2)

Berikut adalah daftar lengkap layanan AI yang terintegrasi di dalam Terry saat ini:

## 1. Otak Utama (Thinking AI)
Urutan prioritas Terry dalam memilih otak:
1.  **Google Gemini** (Model: `gemini-2.5-flash`, `gemini-3-flash`, `gemini-2.0-flash`, dll)
    -   Status: ✅ AKTIF (Auto-Fallback).
    -   Fungsi: Otak utama. Terry akan otomatis mencari versi Gemini yang tersedia.
    -   Update: Menggunakan logika deteksi otomatis untuk menghindari error 404.
2.  **Perplexity AI** (Model: `sonar`, `sonar-pro`)
    -   Status: ✅ AKTIF (Jika API Key ada).
    -   Fungsi: Riset mendalam, berita terkini, dan akses data real-time.
    -   Warning: Model `sonar-reasoning` sudah tidak tersedia (deprecated).
3.  **DeepSeek** (Model: `deepseek-chat`)
    -   Status: ⚠️ ERROR (Perlu Saldo/Top-up).
    -   Fungsi: Otak cadangan jika Gemini & Perplexity mati.
5.  **Ollama (Local AI)** (Model: `llama3.2`, `deepseek-r1:7b`)
    -   Status: ✅ AKTIF (Local - Offline).
    -   Fungsi: Otak lokal tanpa internet & kuota (100% Gratis).
    -   Kelebihan: Sangat cepat (Llama 3.2) & Cerdas (DeepSeek).
6.  **Google Search** (Fallback Terakhir)
    -   Status: ✅ AKTIF.
    -   Fungsi: Jika semua AI mati (termasuk lokal), Terry akan membuka browser Google Search.

## 2. Visual & Mata (Image & Vision)
1.  **Pollinations AI**
    -   Status: ✅ AKTIF (Gratis).
    -   Fungsi: Membuat gambar (Text-to-Image) tanpa login.
    -   Trigger: "Buatkan gambar...", "Lukiskan..."
2.  **Hugging Face Inference API**
    -   Status: ✅ SIAP (Backup).
    -   Fungsi: Menggantikan Pollinations jika down (perlu `HF_TOKEN`).
3.  **Local Web Capture**
    -   Status: ✅ AKTIF.
    -   Fungsi: Screenshot website (`pyautogui` + `webbrowser`).
    -   Trigger: "Screenshot website..."

## 3. Pendengaran & Suara (Audio)
1.  **Speech Recognition** (Google Speech API)
    -   Status: ✅ AKTIF.
    -   Fungsi: Mengubah suara user menjadi teks (STT).
2.  **Edge TTS** (Microsoft Edge Text-to-Speech)
    -   Status: ✅ AKTIF.
    -   Fungsi: Suara Terry (Bahasa Indonesia - Gadis).

## 4. Pencarian (Search)
1.  **DuckDuckGo Search**
    -   Status: ✅ AKTIF.
    -   Fungsi: Mencari info cepat (berita, harga, lirik) untuk memperkaya jawaban AI.
    -   Trigger: "Carikan...", "Harga...", "Siapa..."

---
**Catatan:**
- Semua file konfigurasi ada di `.env`.
- Log aktivitas bisa dilihat di Web UI (Terminal Info).
